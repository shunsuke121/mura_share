# MURAシェア 操作マニュアル（ユーザー向け）

## 1. 目的・対象
MURAシェアを利用する出品者・購入者（借り手）の基本操作をまとめたマニュアルです。

## 2. 画面構成（主なメニュー）
- ホーム（商品一覧）
- 商品を投稿
- レンタル管理
- 購入管理
- 返品管理
- メッセージ
- プロフィール
- 会社概要 / お問い合わせ
- 配送管理（管理者のみ）

## 3. アカウント操作
### 新規登録
1. サイドバーの「ログイン」画面へ進みます。
2. 「新規登録」からユーザー名とパスワードを登録します。

### ログイン / ログアウト
- ログイン後、サイドバー下部にアカウント情報と「ログアウト」ボタンが表示されます。

## 4. プロフィール設定
1. サイドバーの「プロフィール」を開きます。
2. 表示名、電話番号、住所、プロフィール画像を登録します。
3. 住所が未登録の場合、商品投稿や申請時に案内が表示されます。

## 5. 商品一覧（ホーム）
- 検索、カテゴリ、提供方法、並び替えで絞り込みできます。
- ハートアイコンでお気に入り登録/解除できます。
- 売り切れ（sold out）は購入・レンタル・コメントができません。

## 6. 商品詳細での申請（レンタル/購入）
1. 商品一覧から商品を開きます。
2. 申請種別（レンタル / 購入）を選択します。
3. 個数、期間（レンタルのみ）、配送先、支払い方法を入力します。
4. 必要に応じてメッセージを入力し、「申請」します。

補足:
- 郵便番号入力で住所を自動補完します。
- 支払い方法に応じて追加入力欄が表示されます。

## 7. 商品を投稿（出品者）
1. サイドバーの「商品を投稿」を開きます。
2. 画像、商品名、説明、カテゴリ、状態、在庫数を入力します。
3. 提供方法（レンタルのみ / 販売のみ / 両方）と価格を入力します。
4. 「投稿」を押して出品します。

## 8. レンタル管理
レンタル管理は「私の申請」と「受け取った申請」に分かれます。

### 私の申請（借り手）
- 申請中はキャンセルできます。
- 「発送済み」になったら「受け取り」で受領を完了します。
- レンタル中は返送用の追跡番号を入力して「返送」します。
- レンタル中に購入可能な商品は「購入する」から購入へ移行できます。
  - 購入画面で「購入価格 - レンタル利用分」が確認できます。
- 完了した取引は「非表示」にできます（取引履歴から閲覧可能）。

### 受け取った申請（出品者）
- 申請が届いたら「承認」または「却下」します。
- 承認後は追跡番号を入力して「発送」します。
- 返送が届いたら「返却受取（受領）」で完了します。
- 完了した取引は「非表示」にできます（取引履歴から閲覧可能）。

## 9. 購入管理
購入管理は「私の購入」と「受け取った申請」に分かれます。

### 私の購入（購入者）
- 申請中はキャンセルできます。
- 承認/発送後は「受取完了」で取引を完了します。
- 完了した取引は「非表示」にできます。

### 受け取った申請（出品者）
- 申請が届いたら「承認」します。
- レンタルからの購入は承認のみで完了扱いになります。
- 通常の購入は追跡番号を入力して「発送」します。
- 完了した取引は「非表示」にできます。

## 10. 返品管理
### 購入者側
- 「返品申請」を行い、理由（任意）を入力します。
- 承諾後、追跡番号を入力して「返送」します。

### 出品者側
- 返品申請を「承諾」または「却下」します。
- 返送が届いたら「受領完了」を押します。

完了した返品は「非表示」にできます。

## 11. メッセージ（チャット）
1. サイドバーの「メッセージ」を開きます。
2. 取引ごとのチャットルームを選択します。
3. メッセージを入力して送信します。

補足:
- 売り切れの商品はコメント/チャットができません。

## 12. 通知
チャットや取引の更新は「通知」から確認できます。

## 13. 取引履歴
完了・非表示にした取引はプロフィールの「取引履歴」から確認できます。

## 14. 会社概要 / お問い合わせ
- 会社概要は基本情報の確認ページです。
- お問い合わせフォームから送信できます（添付ファイル対応）。

## 15. 配送管理（管理者のみ）
管理者アカウントは「配送管理」から発送状況を確認できます。
